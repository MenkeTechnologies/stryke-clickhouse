What it is

stryke-clickhouse is a ClickHouse client for stryke. It talks to ClickHouse's HTTP interface (default port 8123) and ships as a Rust cdylib dlopened in-process on the first use Clickhouse. Transport is ureq over rustls — synchronous, pure-Rust, no tokio. A keep-alive connection pool is cached per (base_url, auth, database).

Install

s add github.com/MenkeTechnologies/stryke-clickhouse

Quick start

use Clickhouse

var %conn
$conn{url} = "http://127.0.0.1:8123"

Clickhouse::create_table("events", columns => "ts DateTime, user UInt64, kind String", order_by => "(ts, user)", %conn)
Clickhouse::insert("events", [{ ts => "2026-06-20 12:00:00", user => 1, kind => "click" }], %conn)

p Clickhouse::count("events", %conn)
p Clickhouse::query_value("SELECT uniqExact(user) FROM events", %conn)
